# Upload Steps

1. Create a new GitHub repository named still-salt-command-center.
2. Set it to Private.
3. Do not add README, .gitignore, or license on creation.
4. Download this ZIP and unzip it.
5. Open the unzipped folder.
6. Drag the CONTENTS of the folder into GitHub upload, not the ZIP file itself.
7. Commit to main.
8. Add ANTHROPIC_API_KEY in GitHub Actions secrets.
9. Create a new GitHub issue titled ORCHESTRATOR: Build Still Salt Command Center.
10. Paste the contents of CLAUDE_ORCHESTRATOR_PROMPT.md into the issue.
11. Wait for audit plan before approving edits.
